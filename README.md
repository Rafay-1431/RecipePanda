RecipePanda is a responsive web application that allows users to browse through a collection of delicious recipes.
---
Key Features:

Dynamic Rendering: Recipes are generated dynamically using JavaScript from a data object/array.

Detailed View: Users can click on any recipe to see its full details, including a high-quality image, ingredients list, and step-by-step instructions.

State Management: Uses localStorage to pass data between the home page and the details page efficiently.

Modern UI/UX: Clean, card-based layout with a custom-designed scrollbar and hover effects.

Fully Responsive: Works perfectly on desktops, tablets, and mobile devices.
---
Tech Stack:

HTML5 (Semantic structure)

CSS3 (Flexbox, custom animations, and UI styling)

JavaScript (ES6+) (DOM manipulation, loops, and local storage)
